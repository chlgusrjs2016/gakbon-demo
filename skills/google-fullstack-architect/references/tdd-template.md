# Technical Design Template

Use this template when producing design docs.

## 1) Problem and Scope

- Goal
- In scope
- Out of scope
- Assumptions

## 2) Requirements

- Functional requirements
- Non-functional requirements (latency, availability, security)

## 3) Architecture

- Component diagram description
- Request lifecycle
- Failure modes

## 4) Data Model

- Tables and relationships
- Index strategy
- Migration strategy

## 5) API Contract

- Endpoint table
- Request/response examples
- Error model

## 6) Security

- AuthN/AuthZ
- Secrets and key management
- Data protection

## 7) Reliability and Performance

- SLOs
- Capacity assumptions
- Load and benchmark plan

## 8) Delivery Plan

- Weekly sprint plan
- Risks and mitigations
