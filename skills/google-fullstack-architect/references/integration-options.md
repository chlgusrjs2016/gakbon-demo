# Third-party Integration Options

## Payments

- Stripe: best developer ecosystem and docs.
- Paddle: useful for MoR/tax handling.

## Email

- Resend: simple developer DX.
- SendGrid: broad enterprise support.

## Storage

- S3-compatible object storage.
- Supabase Storage for integrated Postgres projects.

## Analytics

- Product analytics: PostHog, Amplitude.
- Web analytics: GA4.

## Selection Heuristic

Choose provider by:

1. Regulatory and region requirements
2. Pricing at expected scale
3. SDK maturity and operational overhead
4. Required feature depth
